# Execution Plans

Long-lived plans for complex work in this repository.

- Put active work in [`active/`](active/).
- Move finished plans to [`completed/`](completed/).
- Start from [`templates/execution-plan.md`](templates/execution-plan.md).
- Record deferred cleanup in [`tech-debt-tracker.md`](tech-debt-tracker.md).

See [`../PLANS_GUIDE.md`](../PLANS_GUIDE.md) for when to create a plan.
